# 生词收藏夹

在网页里读英文时，顺手收藏那些“不认识但感觉下次还会遇到”的单词，并自动附上中文含义。

这个扩展是给一种很具体的阅读瞬间做的：你正在认真读网页，突然撞见一个生僻词；你想查一下、记一下，但一拿起手机，二十分钟后已经在完全无关的地方漫游。生词收藏夹把这条小路径留在浏览器里，让你不用离开当前页面。

使用方式很短：

1. 在网页里选中一个英文单词。
2. 点一下浮出的 `收藏`。
3. 之后打开扩展，就能看到单词、中文含义、原句上下文和来源网页。

## 功能

- 在任意网页中选中英文单词后直接收藏
- 尽可能自动补上中文含义
- 保存单词所在句子，方便回到真实语境里理解
- 保存来源网页标题和链接
- 支持按单词、中文含义、上下文或来源网页搜索
- 可以标记为 `学习中` 或 `已记住`
- 可以删除条目
- 可以导出 CSV
- 数据保存在浏览器本地扩展存储中

## 安装

### 从发布包安装

1. 在 GitHub 发布页下载 `rare-word-keeper-v0.1.1.zip`。
2. 解压这个文件。
3. 打开 `chrome://extensions` 或 `edge://extensions`。
4. 开启“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后的文件夹。

### 从源码安装

1. 克隆或下载这个仓库。
2. 打开 `chrome://extensions` 或 `edge://extensions`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择项目文件夹。

如果你更新了扩展文件，需要在浏览器扩展管理页点击“重新加载”，并刷新正在阅读的网页。

## 使用

在网页里选中一个英文单词，旁边会出现一个小小的收藏按钮。点一下，这个词就会进入你的生词列表。

点击浏览器工具栏里的扩展图标，可以查看收藏的单词、搜索、标记已记住、删除，或把全部记录导出为 CSV。

## 隐私

你收藏的单词会保存在浏览器的本地扩展存储中。

为了补充中文含义，扩展会把你选中的单个英文词发送给在线翻译接口。它不会发送你的完整生词列表、浏览历史、整页文本、个人文件或设备信息。

## 项目结构

```text
.
├── background.js      # 处理翻译请求
├── content.js         # 监听页面选词并显示收藏按钮
├── content.css        # 页面内收藏按钮样式
├── popup.html         # 扩展弹窗界面
├── popup.css          # 扩展弹窗样式
├── popup.js           # 生词列表、筛选、导出逻辑
└── manifest.json      # 浏览器扩展配置
```

## 发布

这是一个普通的 Manifest V3 扩展，不需要构建步骤。

本地生成发布包：

```bash
node scripts/package-release.js
```

生成的压缩包会放在 `release/` 目录中，可以直接上传到 GitHub 发布页。压缩包根目录就是扩展文件，用户解压后可以直接通过“加载已解压的扩展程序”安装。
