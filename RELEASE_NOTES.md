# 生词收藏夹 0.1.1

这是第一个适合直接使用的发布版本。

## 这版有什么

- 在网页中选中英文单词后，可以直接收藏。
- 收藏时会尽可能自动补上中文含义。
- 会保存单词所在句子的上下文。
- 会保存来源网页标题和链接，之后可以回到原文。
- 扩展弹窗里可以搜索、删除、标记 `学习中` 或 `已记住`。
- 可以把全部记录导出为 CSV。

## 安装方式

1. 下载 `rare-word-keeper-v0.1.1.zip`。
2. 解压。
3. 打开 `chrome://extensions` 或 `edge://extensions`。
4. 开启“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后的文件夹。

## 隐私说明

收藏数据保存在浏览器本地。为了补充中文含义，扩展会把你选中的单个英文词发送给在线翻译接口；不会发送完整网页、浏览历史、个人文件或设备信息。
